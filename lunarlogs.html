Lunar Logs

HTML/CSS/JS & PHP/MYSQL

objectives

Lunar Logs is a database-driven web application that hosts up-to-date moon data and personal horoscopes by use of data collection and storage with MYSQL. The site grabs locational data from the user, and uses Skyfield to display accurate astronomical data. With the user's identificational data, data is taken from AstroDataBank for horoscopes.
